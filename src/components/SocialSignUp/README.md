### Usage

```
fronthack component SocialSignUp
```

#### JSX example:

```jsx
<SocialSignUp label='Log in using social media account' />
```

##### Properties

* `variant` - style variant
* `size` - button sizing
* `onClick` - callback after click
* `disabled` - should button be active or not
