# Loader

### Usage

```
fronthack component Loader
```

#### JSX example:

```jsx
<Loader
  loading={true}
  error={false}
>Asynchronic content</Loader>
```

##### Properties

* `loading` - a boolean that indicates loading state
* `error` - a boolean that indicates error state
