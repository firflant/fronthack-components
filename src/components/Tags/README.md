# Tags

### Usage

```
fronthack component Tags
```

#### JSX example:

```jsx
<Tags tags={['travel', 'nature', 'people']} />
```

##### Properties

* `tags` - array of tag labels


#### Static HTML example:

```html
<div class="tags">
  <span class="tags__tag">travel</span>
  <span class="tags__tag">nature</span>
  <span class="tags__tag">people</span>
</div>
```
