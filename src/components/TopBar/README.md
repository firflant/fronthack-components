# Top Bar

### Usage

`fronthack component`, then `top-bar`

#### JSX example:

```js
<TopBar links={[
  { label: 'Projects', url: '/projects' },
  { label: 'About us', url: '/about' },
  { label: 'Contact', url: '/contact' },
]}>
```

##### Properties

* `links` - objects, where each contains two sting values - label and url
