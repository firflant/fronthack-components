# Info Box

### Usage

```
fronthack component InfoBox
```

#### JSX example:

```jsx
<InfoBox
  text='An important text to show.'
  button={<Button size='lg'>Button</Button>}
/>
```

##### Properties

* `text` - a text to show
* `button` - button code


#### Static HTML example:

```html
<div className="info-box">
  <div className="info-box__text">An informational text</div>
  <a href="#" class="btn">Button</a>
</div>
```
